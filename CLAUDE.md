# Jake Walkden — Portfolio

## Project Brief
Personal portfolio website for a 3D artist transitioning into front-end web development.

## Aesthetic
PS2 memory card save screen. Deep black background, teal glow, neon accents.
Grid of "save slots" — each one represents a 3D art project.
Hovering a slot reveals project details in a panel below the grid.
Icons will eventually be pre-rendered 3D turntable videos (WebM/MP4).

## Stack
Plain HTML, CSS, JavaScript — no frameworks.

## Build Order
1. HTML foundation and structure
2. CSS variables and base styles
3. Save slot grid layout
4. Hover interactions and detail panel
5. Animations and glow effects
6. Swap in real project data and video icons

## Key Details
- Title: "Jake Walkden // Portfolio"
- Colour palette: deep black (#050a0f), teal glow (#00ffcc), neon accents
